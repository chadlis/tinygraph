# tinygraph

A minimalist implementation of LangGraph-like core.

## Why?

Understanding LangGraph deeply by rebuilding it.

## Install
```bash
git clone https://github.com/chadlis/tinygraph
cd tinygraph
uv pip install -e ".[dev]"
```
